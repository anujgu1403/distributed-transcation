#Steps to run the java editor api

1. checkout the code from git hub
https://github.com/anujgu1403/distributed-transcation/tree/master/java-editor-api

2. Deploy the code and expose the 8025

3. Change the jdk path to local system path in properties file

4. Post successful deployment access API using below url
http://hostname:8025/api/java/editor

5. provide program as String input in request body then call the api.
